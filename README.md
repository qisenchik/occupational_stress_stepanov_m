# Цифровые технологии как инструмент модернизации исследований иероглифических текстов
## Корпусно-дискурсивное исследование феномена «Тан Пин» (躺平) на материале Bilibili (2020–2025)

Исследование по проекту «Цифровые технологии как инструмент модернизации исследований иероглифических текстов».  
Автор: Степанов Максим Романович, ОП "Востоковедение", НИУ ВШЭ, 2026.

---

## 🔍 Цель исследования
Эмпирический анализ дискурсивных практик репрезентации трудового стресса и культурного феномена «Тан Пин» (躺平 — «лёжа», добровольный отказ от карьерной гонки) в китайской цифровой среде на примере видеоблогов платформы Bilibili.

---

## 📊 Данные
| Параметр | Значение |
|----------|----------|
| Источник | Bilibili (https://www.bilibili.com) |
| Период | Январь 2020 — Декабрь 2025 |
| Объём | 392 видео, 180 304 токена, 49 556 уникальных слов |
| Язык | Китайский (мандарин) |
| Формат | `corpus.csv` / `corpus_clean_nogames.csv` |

**Ключевые запросы для сбора**:  
`躺平`, `内卷`, `996`, `焦虑`, `打工人`, `压力`, `加班`, `摆烂`, `佛系` и др.

## ⚙️ Методы и пайплайн
Проект реализует 5-этапный аналитический пайплайн (`pipeline.py`):

```mermaid
flowchart LR
    A[Сбор метаданных через Bilibili API] --> B[Очистка и токенизация jieba]
    B --> C[Фильтрация игрового/технического шума]
    C --> D[Частотный анализ + KWIC + коллокации]
    D --> E[TF-IDF для отбора релевантных видео]
    E --> F[LDA: тематическое моделирование 6 тем]
